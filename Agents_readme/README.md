# 分子动力学模拟文档指引

## 文档概述

本文档给出了许多在进行分子动力学模拟时可以参考的文档，当你需要的时候你可以使用工具访问对应的文档。

## 知识文档清单
- **[protein.md](./protein.md)** 蛋白质预处理的专业知识,当你在蛋白质预处理时遇到问题可以参考它。
- **[ligand.md](./ligand.md)** 配体参数化的专业知识,当你在配体处理时遇到问题可以参考它。
- **[ligand_tools.md](./ligand_tools.md)** 配体参数化的工具介绍，当你配体处理遇到问题的时候可以参考它。
- **[set_env.md](./set_env.md)** 配置环境的专业知识文档，当你需要配置环境时可以阅读参考它。
- **[dock.md](./dock.md)** 分子对接的专业知识文档，当你分子对接的时候可以阅读参考它。
