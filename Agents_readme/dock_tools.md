# 分子对接工具使用指南

## 对接相关知识
详细的分子对接理论、流程和标准请参阅 **[dock.md](./dock.md)**。

## 文档概述
本指南旨在解决使用 AutoDock Vina 对接时活性位点选取和盒子参数设置的困难。通过集成以下工具实现全自动对接流程：

- **P2Rank**：基于机器学习的结合口袋预测，自动识别蛋白质表面潜在活性位点。
- **GetBox-PyMOL-Plugin**：在 PyMOL 中根据口袋残基精确计算对接盒子（中心坐标和尺寸）。
- **AutoDock Vina**：执行分子对接，预测配体结合构象和亲和力。

## 工具使用
- get_docking_box_from_p2rank -一个高度封装的工具，输入蛋白质预处理后pdb的地址，即可获取活性位点坐标和对接盒子大小，可供对接使用。
